Гадание "Морской бой"

- одиночное попадание

- одиночное совпадение

- двойное совпадение

R - кнопка сброса всех данных с поля

N - кнопка сброса числовых данных с поля

S - кнопка начала расчетов

Правила:

Правый столбец заполняем желаемыми значениями.

Верхнюю и нижнюю строчку, а также левый столбец заполняем цифрами от 1 до 10, щелкая для этого по ячейкам в желаемой последовательности.

После заполнения всех полей - нажимаем кнопку старт и наслаждаемся результатом.

Для нового гадания с уже введенными значениями нужно нажать кнопку ResetNumber

Для совсем нового гадания нужно нажать кнопку Reset
